Funções das tags

● TABLE: A tag < table > representa uma tabela em HTML. Dentro do elemento < table > < /table >, devemos utilizar todos os elementos HTML que compõem as células da tabela.
● TH: Cada cabeçalho da tabela é representado pela tag < th >. Por padrão, o texto nos elementos  < th > está em negrito e centralizado.
● TR: Cada linha de uma tabela é representada pela tag  < tr >.
● TD: Cada célula da tabela é representada pela tag < td >. Por padrão, o texto nos elementos < td > é regular e alinhado à esquerda.
● THEAD: Podemos usar a tag < thead > - para representar o cabeçalho da tabela. 
● TBODY: Podemos usar a tag < tbody > - para representar  o corpo da tabela.
● TFOOT: Podemos usar a tag < tfoot > - para representar  o rodapé da tabela.
● ALIGN: O atributo aling especifica o alinhamento horizontal do conteúdo da célula.
● BORDER: O atributo border adiciona uma borda na tabela como um todo. Borda é uma linha que fica ao redor de algum elemento para destacá-lo.
● CAPTION: A tabela HTML tem uma tag específica para definir um título, é a tag < caption >, ou seja, podemos dizer que a tag < caption > é a legenda da tabela.
● ROWSPAN: O valor do atributo rowspan é o número de linhas que você quer que sejam ocupadas.
● COLSPAN: O atributo colspan permite que uma célula ocupe várias colunas, ou seja, permite mesclar colunas.
● WIDTH: O atributo width especifica a largura da tabela.
